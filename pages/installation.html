Kurulum Rehberi

Kali Linux'u sisteminize kurmak için adım adım talimatlar.

Sistem Gereksinimleri

Kali Linux'u sorunsuz çalıştırmak için minimum donanım gereksinimleri.

İşlemci Intel Core i3 / AMD Ryzen 3
RAM Minimum 2 GB (Önerilen: 4 GB+)
Disk Alanı Minimum 20 GB (Önerilen: 40 GB+)
İnternet Güncelleme ve paket kurulumu için gerekli

Kali Linux'u İndirin

İhtiyacınıza uygun Kali Linux sürümünü seçin.

Kali Linux Installer

Full kurulum için standart ISO. Tüm araçları içerir.

~4.2 GB

Kali Linux Live

USB'den çalıştırmak için canlı ISO. Kurulum gerektirmez.

~3.8 GB

ARM Sürümü

Raspberry Pi ve diğer ARM cihazlar için özel yapı.

ARM64

Sanal Makine

VMware ve VirtualBox için hazır sanal makine imajı.

VM/VMware

Windows WSL

Windows Subsystem for Linux üzerinde çalıştırma.

WSL

Cloud

AWS, Azure ve GCP için hazır cloud imajları.

Cloud

Kurulum Adımları

Kali Linux'u bilgisayarınıza kurmak için aşağıdaki adımları takip edin.

1

ISO Dosyasını İndirin

Yukarıdaki indirme kartlarından ihtiyacınıza uygun Kali Linux ISO dosyasını indirin. Çoğu kullanıcı için Kali Linux Installer önerilir.

2

Önyüklenebilir USB Oluşturun

İndirdiğiniz ISO dosyasını kullanarak önyüklenebilir bir USB sürücü oluşturun. Bunun için Rufus (Windows), Etcher veya dd komutunu kullanabilirsiniz.

Linux/macOS 
$ sudo dd if=kali-linux.iso of=/dev/sdb bs=4M status=progress
3

USB'den Boot Edin

Bilgisayarınızı yeniden başlatın ve BIOS/UEFI ayarlarından USB sürücüden boot etmeyi seçin. Genellikle F12, F2 veya ESC tuşu ile boot menüsüne erişebilirsiniz.

4

Kurulum Dilini Seçin

Kali Linux kurulum sihirbazı açılacaktır. İlk adımda tercih ettiğiniz dili seçin. Türkçe veya İngilizce önerilir.

5

Ağ Ayarlarını Yapılandırın

Kurulum sırasında ağ arayüzünüzü yapılandırmanız istenecektir. DHCP otomatik yapılandırma için genellikle yeterlidir.

6

Disk Bölümleme

Kali Linux'un kurulacağı disk bölümünü seçin. Yeni başlayanlar için "Guided - use entire disk" seçeneği önerilir. İleri düzey kullanıcılar manuel bölümleme yapabilir.

Not: Disk bölümleme sırasında tüm veriler silinecektir. Önemli dosyalarınızı yedeklemeyi unutmayın.

7

Kullanıcı ve Parola Belirleyin

Root (yönetici) parolasını belirleyin. Güçlü bir parola seçmeye özen gösterin. İsteğe bağlı olarak normal bir kullanıcı hesabı da oluşturabilirsiniz.

8

Kurulumu Tamamlayın

Kurulum tamamlandıktan sonra sistemi yeniden başlatın. USB sürücüyü çıkarın ve Kali Linux'a giriş yapın. İlk girişte aşağıdaki komutlarla sistemi güncelleyin:

Terminal 
$ sudo apt update
$ sudo apt full-upgrade -y
$ sudo reboot

İlk Kurulum Sonrası Yapılacaklar

Kali Linux kurulumundan sonra yapmanız önerilen temel ayarlar.

Sistemi Güncelleyin

Tüm paketleri en son sürüme güncelleyin.

Terminal
$ sudo apt update && sudo apt full-upgrade -y

Ek Araçlar Kurun

İhtiyacınıza göre ek güvenlik araçları kurabilirsiniz.

Terminal
$ sudo apt install -y kali-linux-large
$ sudo apt install -y kali-tools-top10

Güvenlik Duvarını Yapılandırın

Varsayılan güvenlik duvarı kurallarını kontrol edin.

Terminal
$ sudo ufw enable
$ sudo ufw status verbose

Snapshot Alın

Sanal makine kullanıyorsanız, temiz bir kurulumun snapshot'ını alın. Böylece herhangi bir sorunda hızlıca geri dönebilirsiniz.

İpucu: VirtualBox veya VMware'de snapshot almak için Machine > Take Snapshot menüsünü kullanın.
