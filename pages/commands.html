Ana Sayfa Temel Komutlar

Temel Komutlar

Kali Linux'da sıkça kullanılan temel terminal komutları ve açıklamaları.

Ağ Komutları

Ağ yapılandırması, bağlantı testi ve izleme için kullanılan komutlar.

Komut Açıklama Kategori
ip a Tüm ağ arayüzlerini ve IP adreslerini gösterir Ağ
ip link set eth0 up/down Ağ arayüzünü aktif/pasif yap Ağ
ping -c 4 8.8.8.8 Hedefe ICMP paketi göndererek bağlantı testi yapar Ağ
traceroute hedef.com Hedefe giden yoldaki tüm yönlendiricileri gösterir Ağ
netstat -tulpn Dinlemedeki portları ve bağlı servisleri listeler Ağ
ss -tulwn Netstat'a alternatif, soket istatistiklerini gösterir Ağ
curl -I hedef.com HTTP header bilgilerini alır Ağ
wget http://hedef.com/dosya Web'den dosya indirir Ağ
dig hedef.com DNS sorgusu yapar Ağ
nslookup hedef.com DNS adres çözümlemesi yapar Ağ

Sistem Komutları

Sistem bilgisi, işlem yönetimi ve kaynak kullanımını gösteren komutlar.

Komut Açıklama Kategori
uname -a Tüm sistem bilgilerini gösterir (kernel, mimari, OS) Sistem
cat /etc/os-release İşletim sistemi sürüm bilgisini gösterir Sistem
htop Gelişmiş işlem yöneticisi (etkileşimli) İşlem
ps aux Tüm çalışan işlemleri listeler İşlem
kill -9 PID Belirtilen PID'ye sahip işlemi zorla sonlandırır İşlem
free -h RAM kullanımını insan tarafından okunabilir formatta gösterir Sistem
df -h Disk bölümlerinin kullanımını gösterir Disk
du -sh * Dizinlerin boyutunu gösterir Disk
uptime Sistemin ne kadar süredir çalıştığını gösterir Sistem
dmesg Kernel mesajlarını gösterir (donanım hataları için) Sistem

Dosya İşlemleri

Dosya ve dizin yönetimi için temel komutlar.

Komut Açıklama Kategori
ls -la Dosyaları detaylı listeler (gizli dosyalar dahil) Dosya
cd /dizin/yolu Belirtilen dizine geçiş yapar Dosya
cp kaynak hedef Dosya veya dizin kopyalar Dosya
mv kaynak hedef Dosya veya dizin taşır / yeniden adlandırır Dosya
rm -rf dizin Dizini ve içindeki her şeyi siler (dikkatli kullanın) Dosya
mkdir yeni_dizin Yeni bir dizin oluşturur Dosya
touch dosya.txt Boş bir dosya oluşturur veya zaman damgasını günceller Dosya
cat dosya.txt Dosya içeriğini ekrana yazdırır Dosya
nano dosya.txt Nano editörü ile dosyayı düzenler Dosya
grep "kelime" dosya Dosyada belirtilen kelimeyi arar Dosya

Yetki ve Paket Yönetimi

Kullanıcı yetkileri ve paket yönetimi komutları.

Komut Açıklama Kategori
sudo komut Root yetkileriyle komut çalıştırır Yetki
sudo -i Root shell'ine geçiş yapar Yetki
chmod 755 dosya Dosya yetkilerini değiştirir Yetki
chown kullanici:grup dosya Dosya sahibini değiştirir Yetki
adduser kullanici Yeni kullanıcı oluşturur Yetki
usermod -aG sudo kullanici Kullanıcıyı sudo grubuna ekler Yetki
apt update Paket listesini günceller Paket
apt install paket Yeni paket kurar Paket
apt remove paket Paketi kaldırır (konfigürasyon dosyalarını silmez) Paket
apt autoremove Gereksiz paketleri temizler Paket

Kali Linux'a Özel Komutlar

Sadece Kali Linux'da bulunan veya sık kullanılan özel komutlar.

Komut Açıklama Kategori
kali-tools-top10 En popüler 10 Kali aracını listeler/kurar Kali
kali-tools-list Kategoriye göre araçları listeler Kali
airmon-ng Wi-Fi arayüzünü monitor mode'a alır Kali
msfconsole Metasploit Framework konsolunu başlatır Kali
service postgresql start PostgreSQL veritabanını başlatır (Metasploit için) Kali
leafpad Hafif bir metin düzenleyici açar Kali
sherlock kullanici_adi OSINT ile kullanıcı adını sosyal medyada arar Kali
hash-identifier Hash türünü tanımlamaya yardımcı olur Kali
